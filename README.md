# Pimcore PHP-FPM Image

## Supported tags and respective Dockerfile links
 - PHP7.1
    - cli
    - apache
    - fpm
 - PHP7.2
    - cli
    - apache
    - fpm
 - PHP7.3
    - cli
    - apache
    - fpm

## Debug Images
Images suffiex with debug have xdebug installed. You can deactivate xdebug with env variable `PHP_DEBUG=0`.
